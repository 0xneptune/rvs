# rvs

Risky vs Safe (**RVS**) decision making game for the [Computational Affective Neuroscience (CAN) Lab]. In addition, some data analysis for **MES** (**Moral Evaluation
Study**).

### Requirements

- Inquisit 5
- R 3.3.2
